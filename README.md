Binary conversion
Samuel Morse and Alfred Vail created Morse code in the 1830s–1840s for use with the telegraph system. It was designed to send messages quickly over long distances before modern phones existed.

Morse code works by turning letters and numbers into patterns of short and long signals called dots and dashes. For example, the letter “A” is a dot followed by a dash (• —). An operator presses a telegraph key to send electrical pulses through a wire, and the receiver hears or records the pattern on the other side. The listener then translates the patterns back into words.

Because it only needs simple signals, Morse code could also be sent using sound, light flashes, or radio waves. A language built from interruption and silence—strangely elegant for something so mechanical.
